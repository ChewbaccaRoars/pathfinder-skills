# Build Mode

## Overview
A shortcut that bundles six builder skills into one flow. Find what to build, form a hypothesis, get a rough version working fast, and level up from scripts to agents.

**Keywords**: build, create, make, automate, code, script, app, agent, prototype

## What Activates
- **workflow-detective** — find the highest-value thing to automate
- **hypothesis-lab** — structure the project as an experiment
- **build-first-break-later** — get a rough v1 working in minutes
- **apps-script-builder** — for Google Workspace automations (with CLASP, full service reference, patterns)
- **api-explorer** — when you need to connect systems
- **agent-architect** — when you're ready to level up to agents

## Recommended Flow
1. Start with "what should I build?" (workflow-detective)
2. Pick the best opportunity, form a hypothesis (hypothesis-lab)
3. Build a rough v1 fast (build-first-break-later)
4. Use apps-script-builder or api-explorer depending on what you're building
5. When ready, level up to agent-architect

## On Activation
Tell the user:
"Build mode is on. I'll help you find what to build, form a hypothesis, get a rough version working fast, and level up from there. What do you want to create?"

"Say '/reflect' when you're done building, or '/pathfinder' to see all options."

## Tone Guidelines
- Energetic, action-oriented, momentum-focused
- Bias toward building — don't over-plan, start fast
- Celebrate every iteration

Do NOT include personal info.
