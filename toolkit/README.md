# Pathfinder Toolkit

Practical templates for running a Pathfinder tour of duty. Use these to observe, build, transfer, and measure — consistently across every tour.

## Templates

| # | Template | Phase | What it is |
|:--|:---------|:------|:-----------|
| 01 | **[Intake Template](templates/01-intake-template.md)** | Observe | Pre-tour form + 1:1 interview guide + team snapshot |
| 02 | **[Meeting Observer Notes](templates/02-meeting-observer-notes.md)** | Observe | Structured template for shadowing meetings |
| 03 | **[Workflow Mapper](templates/03-workflow-mapper.md)** | Observe | Map workflows, score pain points, find quick wins |
| 04 | **[Solution Registry](templates/04-solution-registry.md)** | All phases | Master list of everything built across all tours |
| 05 | **[Transition Checklist](templates/05-transition-checklist.md)** | Harvest | End-of-tour checklist — don't leave until it's done |
| 06 | **[Tour Retrospective](templates/06-tour-retrospective.md)** | Harvest | Structured reflection: what worked, what didn't, patterns |
| 07 | **[Champion Tracker](templates/07-champion-tracker.md)** | Multiply | CRM for your growing network of builders |
| 08 | **[Impact Calculator](templates/08-impact-calculator.md)** | Build | Estimate ROI before building, compare actuals after |

## How to Use

1. Copy each template into a Google Doc or Google Sheet (depending on the template)
2. Fill them in as you progress through each tour phase
3. The Solution Registry and Champion Tracker are persistent — they grow across all tours
4. Everything else is per-tour — create a fresh copy for each engagement

## Tour Flow

```
Week 1-2: Observe & Learn
  → 01 Intake Template
  → 02 Meeting Observer Notes
  → 03 Workflow Mapper
  → 08 Impact Calculator (estimate)

Week 3-6: Build & Enable
  → 04 Solution Registry (add entries)
  → 08 Impact Calculator (refine estimates)

Week 7-10: Multiply & Transfer
  → 07 Champion Tracker (identify and assess)

Week 11-12: Harvest & Transition
  → 05 Transition Checklist
  → 06 Tour Retrospective
  → 08 Impact Calculator (actual results)
  → 04 Solution Registry (update status)
  → 07 Champion Tracker (update confidence)
```
